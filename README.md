# 🛒 React E-Commerce Project

## 📌 Project Overview
This is a simple **E-Commerce web application** built using **React.js**.  
The project includes multiple pages such as Home, Shop, Blog, Blog Details, Cart, and Favorites.  
It allows users to browse products, add items to cart, and save items to favorites.

---

## 🚀 Features

- 🏠 Home page with project overview
- 🛍️ Shop page with products and offers
- 📰 Blog page for posts/articles
- 📄 Blog Details page for each post
- 🛒 Add to Cart functionality
- ❤️ Add to Favorites functionality
- 📊 Dynamic counters for cart and favorites
- 🧾 Cart page to display selected products
- ⭐ Favorites page to manage saved items
- 💾 Data stored using LocalStorage (no backend)

---

## 🧰 Technologies Used

- React.js
- React Router DOM
- Bootstrap / React Bootstrap
- JavaScript (ES6+)
- HTML5
- CSS3
- LocalStorage API
- Font Awesome Icons
- Sweet Alert
